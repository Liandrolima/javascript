Manual frontend com React

1. Configurar o Ambiente 1.1 Instale o Node.js Baixe o Node.js e instale a versão LTS. Verifique se o Node.js e o npm foram instalados: bash Copiar código node -v npm -v 1.2 Instale o Gerenciador de Pacotes (Opcional) Você pode usar o yarn como alternativa ao npm: bash Copiar código npm install -g yarn 2. Criar um Projeto React 2.1 Crie o Projeto com Create React App Use o comando abaixo para criar o projeto React: bash Copiar código npx create-react-app nome-do-projeto Ou, se estiver usando yarn: bash Copiar código yarn create react-app nome-do-projeto 2.2 Acesse o Diretório do Projeto Após a criação, entre no diretório: bash Copiar código cd nome-do-projeto 3. Estrutura Inicial do Projeto A estrutura do projeto gerada será semelhante a esta: plaintext Copiar código nome-do-projeto/ ├── node_modules/ ├── public/ ├── src/ │ ├── App.css │ ├── App.js │ ├── index.css │ ├── index.js │ └── ... ├── .gitignore ├── package.json ├── README.md └── ... 4. Iniciar o Servidor de Desenvolvimento 4.1 Execute o Projeto Use o comando: bash Copiar código npm start Ou, se estiver usando yarn: bash Copiar código yarn start O React abrirá automaticamente no navegador, geralmente em http://localhost:3000. 5. Produção e Deploy 9.1 Build do Projeto Gere uma versão otimizada para produção: bash Copiar código npm run build 9.2 Hospede o Projeto Use serviços como: GitHub Pages: Instale o pacote para deploy: bash Copiar código npm install gh-pages Configure o package.json: json Copiar código "homepage": "https://usuario.github.io/repositorio", "scripts": { "predeploy": "npm run build", "deploy": "gh-pages -d build" } Faça o deploy: bash Copiar código npm run deploy Netlify ou Vercel: Faça upload ou conecte ao repositório GitHub.
